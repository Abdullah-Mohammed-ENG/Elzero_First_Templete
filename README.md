# Leon Agency - Creative Web Template

A modern, fully responsive landing page for a creative agency. This project was built to master advanced CSS layout techniques and professional project organization.

## 🚀 Overview
This template focuses on clean design and high performance. It was developed as part of a journey to master frontend development, transforming a conceptual design into a functional, pixel-perfect website.

## 🛠 Tech Stack
*   **HTML5**: Structured with semantic elements for better SEO and accessibility.
*   **CSS3**: Leveraged advanced styling techniques, including:
    *   **CSS Grid**: Used for complex layouts like the Services and Portfolio sections.
    *   **Flexbox**: Utilized for navigation alignment and flexible content positioning.
    *   **Custom Variables**: Implemented a global color palette for easy theme management.
    *   **Media Queries**: Engineered a mobile-first responsive design to support all device sizes.
*   **Font Awesome**: Integrated for high-quality scalable vector icons.
*   **Google Fonts**: Optimized typography using the 'Work Sans' font family.

## 📁 Project Structure
The project follows a clean and scalable directory structure:
*   `/css`: Contains `normalize.css` for cross-browser consistency and `Leon.css` for custom styles.
*   `/images`: All optimized graphical assets.
*   `/webfonts`: Local hosting of Font Awesome icon files.
*   `index.html`: The main entry point of the application.

## ✨ Key Features
- **Responsive Layout**: Seamless transition between Desktop, Tablet, and Mobile views.
- **Modern Grid System**: Efficient use of `grid-template-columns` and `minmax` functions.
- **Professional Folder Organization**: Ready for deployment and easy for other developers to navigate.
- **Optimized Performance**: Clean code with minimal overhead.
